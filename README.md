# Space Station 14 Autochemistry


A simple external tool for **automatic chemistry** in *Space Station 14*.


This program assists with automating chemical preparation workflows by simulating or controlling reagent combinations and station chemistry processes.


**Important:** Use of this tool is **strictly prohibited** on servers.  
Do not use it in multiplayer environments, ranked servers, or where automation tools are banned.


## Features


- Automates repetitive chemical mixing tasks
- Helps prototype reagent recipes
- Can speed up chemistry workflows for testing purposes
- Works outside of the game
